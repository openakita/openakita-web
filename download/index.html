下载与安装 OpenAkita

官方支持 Desktop、CLI、源码部署三种方式。你可以从简单到深入逐步切换。

Desktop 安装包（Win / macOS / Linux）

只保留三端最新安装包入口，按你的系统直接下载即可。

版本：加载中... · --
查看全部安装包 Release 说明
Windows加载中... macOS加载中... Linux加载中...

若当前版本缺少某端包，按钮会自动跳转到 Releases 页。

PyPI CLI 安装

适合命令行用户与自动化部署，步骤最短。

# 1) 创建虚拟环境 python -m venv venv # 2) 激活环境（Linux/macOS） source venv/bin/activate # 3) 激活环境（Windows PowerShell） # .\\venv\\Scripts\\activate # 4) 安装 openakita 与常用扩展 pip install openakita[all] # 5) 初始化配置文件 openakita init # 6) 启动交互模式 openakita
查看安装教程

源码安装

适合需要二次开发、定制构建和自托管流程的团队。

# 1) 拉取源码 git clone https://github.com/openakita/openakita.git cd openakita # 2) 创建并激活虚拟环境 python -m venv venv source venv/bin/activate # 3) 以开发模式安装（含 dev 依赖） pip install -e ".[all,dev]" # 4) 准备运行配置文件 cp examples/.env.example .env cp data/llm_endpoints.json.example data/llm_endpoints.json # 5) 初始化并生成目录 openakita init

一键脚本安装

官方脚本来自项目仓库 `scripts/quickstart.*`。

Linux / macOS

curl -fsSL https://raw.githubusercontent.com/openakita/openakita/main/scripts/quickstart.sh | bash

Windows PowerShell

irm https://raw.githubusercontent.com/openakita/openakita/main/scripts/quickstart.ps1 | iex

脚本会自动准备虚拟环境与基础配置，适合第一次落地。
